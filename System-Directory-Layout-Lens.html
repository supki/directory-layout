text :: Prism Layout Layout Text TextSource

Get Text out of the current Layout (if possible) 

>>> layout ^? text
Nothing
>>> layout ^? file "foo" . text
Just "not empty"
>>> layout ^? directory "bar" . file "quux" . text
Just "something"
